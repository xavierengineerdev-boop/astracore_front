# Переиспользуемые компоненты

Чтобы не дублировать разметку и стили на страницах, используйте общие компоненты и стили.

## Компоненты

- **BackButton** — кнопка «Назад» с переходом по истории или fallback.
- **AppDialog** — диалог в едином стиле (тёмный фон, граница). Вместо копирования `PaperProps={{ sx: {...} }}` на каждой странице.
- **CopyableText** — текст, копируемый в буфер по клику (телефон, email в таблицах).

## Общие стили (`@/theme/formStyles`)

- **formFieldSx** — стили полей форм (цвет текста и лейблов для тёмной темы).
- **formFieldSxTall** — то же + увеличенная высота поля (формы создания/редактирования).
- **dialogPaperSx**, **dialogTitleSx** — стили для диалогов.
- **tableCellSx**, **tableCellSecondarySx**, **tableHeadCellSx** — стили ячеек таблиц.

Импорт: `import { formFieldSx } from '@/theme/formStyles'`

## Рекомендации

1. Новые диалоги делайте через **AppDialog** или используйте **dialogPaperSx** в `PaperProps`.
2. Поля форм на тёмном фоне — **formFieldSx** или **formFieldSxTall**.
3. Ячейки «клик — скопировать» в таблицах — **CopyableText** с `onCopy` (например, `() => toast.success('Скопировано')`).
4. При появлении повторяющегося блока UI (фильтры, карточки, строки таблицы) выносите его в компонент в `components/` или в подпапку страницы.
